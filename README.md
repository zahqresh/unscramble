## General info
A simple webapp that let's you unscramble the words. 
	
## Technologies
Project is created with:
* Express
* Simple-spellchecker
* Body-parser
* axios
* anagram
	
## Setup
To run this project, install it locally by cloning the repo or download the zip file using git clone:
```
$ npm intall
$ node app.js
```
